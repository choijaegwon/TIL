# 터치 이벤트 제거

## 예시코드
~~~
VStack {
    LinearGradient(gradient: Gradient(colors: [.black, .white]), startPoint: .top, endPoint: .bottom).opacity(0.6)
        .allowsHitTesting(false)
}
.frame(height: 129)
~~~

## 핵심코드

~~~
.allowsHitTesting(false)
~~~
뷰 뒤에 있는 것이 대신 클릭 된다.
